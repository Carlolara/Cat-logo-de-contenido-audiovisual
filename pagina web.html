Recursos Digitales

Formatos Digitales Utilizados en Contenidos Web

Imágenes (JPG, PNG, SVG)

Imágenes utilizados para ilustrar o complementar el contenido de un sitio web, sirven para mejorar la aparencia visual y comunican mensajes de forma rápida.

Videos (MP4, MOV, AVI)

Integran imagen y sonido para presentar contenido dinámico. su uso normalmente son para tutoriales, blogs o anuncios.

Audio (MP3, WAV)

Archivos de sonido como narraciones o música. Aportan una experiencia auditiva más rica y emocional.

Animaciones (GIF)

Añaden movimiento y dinamismo. Se usan en botones o simplemente para atraer la atención.

Texto (HTML, PDF, DOCX)

Comunican la información principal del sitio. fortalecen la identidad del contenido.

Interactividad (Formularios, Enlaces)

Permiten la acción del usuario. Mejoran la experiencia de navegación y la participación en el sitio web.
